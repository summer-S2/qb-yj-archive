# 개인 설정 — 정윤지

> 이 파일은 나 혼자 사용하는 설정. 팀 공통 컨벤션은 `CLAUDE.md` 참고.

---

## 응답 규칙

- 모든 답변은 **한국어**로 작성
- 주석도 **무조건 한국어**
- 코드, 함수명, 변수명만 영어 사용
- 불필요한 설명 장황하게 하지 말 것
- 불완전한 코드 예시 제공 금지

## 응답 스타일

- 예제 코드는 바로 실행 가능한 **완성형**으로
- 에러 해결은 **원인 + 해결방법** 모두 설명
- 파일 수정 시 연관된 모든 컴포넌트 전수 조사 → 사이드 이펙트 없음 보장

## 코드 컨벤션

- 변수명: `camelCase`, 의미 명확하게 (예: `userName`, `isActive`)
- 함수명: 동사로 시작, 쉬운 단어 사용 (예: `getUserInfo`, `handleClick`)
- Boolean 변수: `isXXX`, `hasXXX`, `shouldXXX`
- Props 인터페이스: `IProps`

## 작업 방식

- 문서 순서: `SPEC.md` 작성 → 검토 → `PLAN.md` 작성 → 검토 → 코드 작업
- 기획 변경 시: `SPEC.md` 수정 → `PLAN.md` 수정 → 코드 수정 (순서 반드시 지킬 것)

## 금지 사항

- 영어로 주석 달지 말 것
- 장황한 설명 금지
- 불완전한 코드 예시 금지
