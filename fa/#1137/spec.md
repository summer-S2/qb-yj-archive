# 투자제안 다중계좌 — #1137

## 개요

FA가 고객에게 투자 제안 알림톡을 발송할 때 지원하는 계좌 타입을 IRP 단건에서 IRP/ISA/연금저축(PSA)/종합위탁(BA) 4종으로 확대.
포트폴리오 발송 화면에서 계좌 타입별 발송 현황을 한눈에 확인하고 선택해 발송할 수 있도록 개편.

---

## 계좌 타입 변경

| 구분 | 내용 |
|------|------|
| as-is | IRP 단건 |
| to-be | IRP / ISA / 연금저축(PSA) / 종합위탁(BA) |

---

## 변경된 화면

### 포트폴리오 발송 탭 (PortfolioSendTab)

**as-is**
- KB증권 카드에 "IRP" 버튼 하나
- 마지막 발송 일시 텍스트 캡션으로 표시

**to-be**
- KB증권 카드 하위에 계좌 타입별 리스트로 펼쳐짐
- 각 항목에 계좌명·계약유형, 최근 발송 포트폴리오명, 발송 일시 표시
- 발송 이력 없으면 "발송 이력이 없습니다" 표시
- 항목 클릭 → 해당 계좌 타입의 포트폴리오 발송 모달 오픈

### 포트폴리오 발송 모달 (PortfolioModal)

- 포트폴리오 상세에 계좌 타입 Chip, 계약 타입 Chip 추가
- 발송 완료 후 `last-invest-strategy-all` 쿼리 invalidate (발송 현황 즉시 갱신)
- 버튼 문구 변경: "자문 계약 링크 발송" → "계약 링크 발송"

### 포트폴리오 상세 (DetailContent)

- 기관명 Chip 옆에 계좌 타입 Chip, 계약 타입 Chip 추가
- 계약 타입: serviceCode 마지막 자리로 판단 (`0` → 자문, `1` → 일임)
