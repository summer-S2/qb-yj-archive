|--------------|---------------|

작업 요청 시 해당 폴더의 아카이브 문서를 참고하고, 위 경로에서 코드 작업을 진행한다.

---

## 브랜치 작업 순서

1. 원격 브랜치 목록 fetch (`git fetch`)
2. 해당 이슈 브랜치 존재 여부 확인
   - 있으면 → 체크아웃
   - 없으면 → `feature/#이슈번호-기능명` 형식으로 새 브랜치 생성
